# 🎵 last's scriber

A modern **song scribing / lyric timestamping tool** built with Electron.  
Designed for fast, precise lyric timing with waveform support and easy export options.

---

## ✨ Features

- 🎧 Load and play audio files (MP3, WAV, FLAC, etc.)
- 📍 Timestamp lyrics in real-time
- 🎚️ Adjustable playback speed
- 🌊 Waveform visualisation (WaveSurfer.js)
- 📝 Export lyrics to multiple formats:
  - `.SRT`
  - `.LRC`
  - `.ASS`
  - `.JSON`
  - `.TXT`
- 📋 Copy to clipboard export options
- ⌨️ Keyboard shortcuts for fast workflow
- 🖥️ Custom frameless UI (modern dark design)

---

## ⌨️ Controls

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `Q` | Stamp lyric timestamp |
| `T` | Move to next lyric |
| `E` | Export lyrics |
| `+ / -` | Change playback speed |
| `Scroll` | Scrub timeline |
| `Ctrl + Scroll` | Zoom waveform |
| `Shift + Scroll` | Fine scrub |
